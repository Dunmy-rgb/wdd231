Individual Project Site Plan

Site Name

Gaming Nexus – This name reflects a central hub or "nexus" where all things gaming converge. It’s short, dynamic, and ideal for branding in the gaming niche.

Optional domain availability: gamingnexushub.com

Site Purpose

This website provides in-depth game reviews, gaming PC build guides, latest news on game releases, and a community space for gamers to share experiences, tips, and builds. The goal is to inform, support, and unite gamers from all skill levels.

Scenarios

  • What’s the best GPU under $400 for modern games?
  • Which upcoming AAA titles are worth preordering?

Color Schema

  • #121212 (Dark Charcoal) – Main background color for a sleek, modern gamer vibe.
  • #1db954 (Neon Green) – Used for CTAs, links, and highlights.
  • #ffffff (White) – Body text color for readability.

Typography

  • Orbitron – Futuristic, digital-style font for headings and hero banners.
  • Open Sans – Clean and readable for body text and article content.
